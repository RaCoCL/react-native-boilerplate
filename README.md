# react-native-boilerplate

# Getting started

## Requirements
Node 10+ 

## Crete new project 
* NOTE: (do this only if you need to start a new project, this repo already have a project called ShoppingApp)

Install expo-cli

> `npm install -g expo-cli`

Start a project
> expo init ShoppingApp

> cd ShoppingApp

Start App

> npm start # or expo start
